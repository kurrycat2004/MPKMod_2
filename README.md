# MPK Mod
[![Discord](https://img.shields.io/discord/819737524372504587?color=7289DA&logo=discord)](https://discord.gg/rSzmsdXsvW)

MPK Mod is a minecraft mod for 1.8 - 1.19 that adds some features for [minecraft parkour](https://www.mcpk.wiki/wiki/Main_Page) to your client. It's a client side only mod, that means it can be used on any server.

## Content
- [Installation](#Installation)

### Installation

To install MPK Mod you need to have [forge](https://files.minecraftforge.net) installed.
To use the Mod you have to download the file `mpkmod-X.X.X-MCX.X.jar` (for example mpkmod-2.0.0-MC1.8.9) from [releases](https://github.com/kurrycat2004/MpkMod/releases) and put it into your `mods` folder which is located in [`.minecraft`](https://minecraft.gamepedia.com/.minecraft).
